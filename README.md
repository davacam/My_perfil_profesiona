# 🌐 Página de Presentación Personal

Este proyecto consiste en el desarrollo de una página web de presentación personal utilizando tecnologías básicas del desarrollo web: **HTML, CSS y JavaScript**.

## 📌 Descripción

La página fue diseñada como un portafolio personal donde se muestra información relevante sobre el perfil profesional, habilidades, proyectos realizados y medios de contacto.

## 🚀 Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (básico)
* GitHub Pages (para despliegue)

## 📂 Estructura del proyecto

```
index.html
```

Todo el diseño (HTML, CSS y JS) está integrado en un solo archivo para facilitar su implementación y despliegue.

## 🎯 Secciones de la página

* **Inicio:** Presentación breve
* **Sobre mí:** Información personal y profesional
* **Habilidades:** Competencias técnicas y blandas
* **Proyectos:** Experiencia en proyectos reales
* **Contacto:** Enlaces a correo, LinkedIn y GitHub

## 🌍 Despliegue
